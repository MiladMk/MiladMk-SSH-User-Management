# MiladMk Panel — نصب و راه‌اندازی

پنل مدیریت کاربران SSH + Sing-box، کاملاً مستقل و اختصاصی.

## نصب یک‌خطی (Ubuntu 24/26 پیشنهادی، Debian هم پشتیبانی می‌شود)
به عنوان **root**:
```bash
bash <(curl -Ls https://raw.githubusercontent.com/MiladMk/MiladMk-SSH-User-Management/main/install.sh --ipv4)
```
نصب‌کننده نسخهٔ PHP مناسب را خودکار انتخاب می‌کند (Ubuntu 20/22 → 8.1، Ubuntu 24/26 → 8.3) و تنظیمات php-fpm لازم برای ساخت کاربر SSH را خودکار اعمال می‌کند.

## دستور مدیریت
```bash
miladmk
```
(دستور `xpanel` هم برای سازگاری کار می‌کند.)

## بروزرسانی
```bash
miladmk   # گزینه 4
```
یا اجرای مجدد دستور نصب. تنظیمات فعلی حفظ می‌شود.

## قابلیت‌های اختصاصی
- **چرخش IP کلودفلر:** تنظیمات ← IP Rotate. با API خودت، لیست IP، بازه زمانی و حالت ترتیبی/تصادفی. کاملاً روی سرور خودت، بدون هیچ سرویس خارجی.
- **هاست سفارشی:** تنظیمات ← عمومی. لینک‌های کاربران را با هاست دلخواه می‌سازد.
- **NPV Tunnel:** در دراپ‌داون هر کاربر (Direct/TLS/Dropbear).
- **برند MiladMk:** لوگو و نام.

## پشتیبانی
فقط سیستم‌های apt-based (Ubuntu پیشنهادی، Debian best-effort). توزیع‌های دیگر رسماً پشتیبانی نمی‌شوند.

## نکته
یک کنترلر با ionCube انکود شده و به PHP ≤ 8.3 نیاز دارد؛ نصب‌کننده همین را رعایت می‌کند.
